# Lab-System-Logs
Technical logs, security research, and AI experimentation. Documenting the intersection of Cybersecurity and local Machine Learning.

### 🖥️ Lab Specifications
- **OS:** CachyOS (Arch Linux) | **Bootloader:** Limine
- **Primary Tools:** Docker (Flatpak), Ollama, VS Code (Continue/Cline),Python, C++, Bash.

### 📂 Directory Structure
- `/security/`: Pentest tool setups and OSINT logs.
- `/ai/`: Local LLM benchmarks and agentic workflow tests.
- `/linux/`: Kernel tweaks, driver fixes, and CachyOS optimizations.
